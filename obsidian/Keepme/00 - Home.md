# KeepMe Knowledge Base

Multi-service SaaS platform for fitness and wellness facility management.

## Repositories

| Repo | Purpose | Stack |
|------|---------|-------|
| `keepme-react-v4` | Frontend | React 19.1, TypeScript 5.8, Vite 7.1 |
| `keepme-v4` | Backend API | Laravel 12, PHP 8.3, MySQL 8, MongoDB 7 |
| `keepme` | Legacy backend | Laravel 8, PHP 7.3/8.0 |
| `keepme-services` | Webhook microservices | Fastify 5, TypeScript, Kafka |
| `keepme-infra` | Infrastructure as Code | Terraform, Kubernetes, Helm |
| `keepme-argo` | GitOps | ArgoCD, Kubernetes manifests |
| `Antares-project` | Multi-service automation | Node.js, Python, MongoDB, React |
